# Fase 1 — Fundación Astro

La aplicación Astro quedó configurada en la raíz del repositorio.

## Creado

- Tokens CSS de tres niveles con temas claro y oscuro.
- `BaseLayout` con metadatos, datos estructurados, tipografías y arranque de tema sin parpadeo.
- Componentes UI base: botones, tarjeta, kicker, selector de tema, logo y slot de medios.
- Enlace de WhatsApp centralizado.
- Trece stubs vacíos, en el orden definido por el plan.

## Ejecutar

```bash
npm run dev
```
